--> A curated Emacs experience

Emacs Live is a carefully curated Emacs config which transforms vanilla Emacs from simply an awesome editor into an intense experience. The key is in the curation. Unlike other awesome Emacs config tools such as the starter-kit and helm, Emacs Live embodies a holistic approach creating an all-round package encorporating many independent dependencies yet ensuring that they all work together seemlessly.


--> Perfect for Clojure hacking

Emacs Live was designed from the ground up to create an insane Clojure hacking experience. It ships with more bang than you can pack into a firework and doesn't need a safety tin: nREPL support, auto completion with fuzzy matching, snippets, docstring lookup, paredit, rainbow parens, the famous cyberpunk colour theme and many more custom modifications. All you need to do is bring along is a version of Leiningen 2.0.


--> Power features

However, Emacs Live isn't just good for Clojure hacking - it's an all-round text manipulation power house. We pack in the amazing undo-tree, a visual kill-ring, gist integration, ace jump mode, expand region, and much, much more.


--> Trivial installation

Either run the script above, or download the latest release, rename it to ~/.emacs.d and launch Emacs. That's all - joy awaits.
